# 🦎 Reptile Monster Cursor

An interactive web animation that transforms the traditional mouse cursor into a dynamic reptile-inspired monster trail. Built with HTML, CSS, and JavaScript, the project creates smooth cursor-following effects, fluid movement animations, and engaging visual interactions for desktop and mobile users.

## ✨ Features

* 🦎 Reptile-inspired monster cursor
* 🎯 Real-time cursor tracking
* ✨ Smooth motion animations
* 🖱️ Interactive mouse movement effects
* 📱 Mobile touch compatibility
* 🌈 Dynamic visual effects
* ⚡ High-performance rendering
* 🎨 Modern responsive design
* 🚀 Lightweight and dependency-free
